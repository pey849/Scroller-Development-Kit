Firstly Navigate to the file called "For TA" and click on "SDK Application.exe" to start the SDK Application.

First thing you need to do is to browse for an Entity by pressing the "Browse XML" button.

After pressing the "Browse XML" Button you will be prompted to select an entity file. Please select desired Entity (XML).
For this tutorial please navigate to the folder "Data/Entities" and select the file "Player.xml". Just ignore the "must be a .png" error message

You will notice that the Entity filename has been updated to be the file name of your selected Entity.

You will also notice that new tabs have been added. Each tab corresponds to a component for a particular Entity (in this case, Player). Please select a Component (tab) to edit.

**Note** Editing datatypes like Color and Vector2 are presented as a textbox. It updates and checks every time text is changed. This is so the user does not enter any invalid values.

You will also notice that protected fields have been disabled.
