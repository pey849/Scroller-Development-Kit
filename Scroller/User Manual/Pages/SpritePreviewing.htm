Click the select button in the Sprite details tab

Load the Image you want to preview, the application will only accept .png files
For this example please navigate to the file "platformer_sprites_pixelized_0.png" in folder "For TA"

You will notice that the sprite has now been loaded

Now enter the required fields (The ones that were left blank in the image below are optional) and click the play button. To pause press the pause button, To stop press the stop button (this will reset the image and start it at the beginning).
For this tutorial, please enter in:
Sprites/Row: 4
Sprite/Column: 2
Animation_Name: Leave as is
X-Index: 0
Y-Index: 0
Number of Frames: 4
Duration: 0.05


Proceed to Step 3: Save XML
